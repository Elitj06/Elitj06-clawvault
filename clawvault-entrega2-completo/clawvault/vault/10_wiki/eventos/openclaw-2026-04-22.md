## Lições aprendidas
- **Sessões competem por roteamento:** Se o MAIN tem sessão ativa para um grupo, o binding da Laura é ignorado
- **`requireMention` funciona em `channels.whatsapp.groups`** mas NÃO em binding/agente nível
- **Evolution API é necessária** para rotear mensagens de grupo para agentes secundários (Laura)
- **API key typo:** quando testar endpoints manualmente, SEMPRE copiar a key exata — um char errado = 401 em tudo
